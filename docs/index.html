Vue 驱动的静态网站生成器

这是一个仿照 VuePress 官网（v2.0.0-beta.67）的演示网站，默认中文语言

快速上手 项目简介

Vue 驱动

以 Markdown 为中心的项目结构，享受 Vue 的开发体验，可以在 Markdown 中使用 Vue 组件，又可以使用 Vue 来开发自定义主题。

渲染机制

打包工具是 Vite（默认）、Webpack；VuePress 会为每个页面预渲染生成静态的 HTML，每个页面被加载时作为 SPA 运行。

主题插件

提供了一个开箱即用的默认主题；灵活的插件API，使得插件可以为你的站点提供许多即插即用的功能。

# 在项目中集成

# 在你的项目中安装
pnpm add -D vuepress@next @vuepress/client@next vue

# 新建一个 markdown 文件
echo '# Hello VuePress' > README.md

# 开始写作
pnpm vuepress dev

# 构建静态文件
pnpm vuepress build

# 在你的项目中安装
yarn add -D vuepress@next

# 新建一个 markdown 文件
echo '# Hello VuePress' > README.md

# 开始写作
yarn vuepress dev

# 构建静态文件
yarn vuepress build

# 在你的项目中安装
npm install -D vuepress@next

# 新建一个 markdown 文件
echo '# Hello VuePress' > README.md

# 开始写作
npx vuepress dev

# 构建静态文件
npx vuepress build

# 插件（补充）

  • 1.markdown 复制在新窗口打开，支持一键复制功能；
  • 2.markdown 增强在新窗口打开，支持代码块分组、任务列表、流程图、幻灯片等功能；
